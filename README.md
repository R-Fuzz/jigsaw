# JIGSAW

**Build**
```
mkdir build 
cd build && make
```

**Replay from constraints files**
```
Command:
./rgd num_of_threads pin_core_start test_dir

Example:
# solve objdump constraints using 8 cores, starting from core 0
./rgd 8 0 objdump 
```
